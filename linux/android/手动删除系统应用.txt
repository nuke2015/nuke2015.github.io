

1.   显示所有应用 
adb shell pm list packages
2.   删除你需要删除的应用adbshell pm uninstall --user 0 +应用的名称
3.   显示软件名称  com.baidu.input_mi
4.   删除百度输入法做示范 输入  
adb shell pm uninstall --user 0 com.baidu.input_mi  
回车,出现Success  表示删除成功
